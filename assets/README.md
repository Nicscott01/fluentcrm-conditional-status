# Assets Directory

This directory contains WordPress.org plugin repository assets.

## Required Assets

### Plugin Icon
- `icon-128x128.png` - 128x128px icon (required)
- `icon-256x256.png` - 256x256px icon (required for retina)

### Plugin Banner
- `banner-772x250.png` - 772x250px banner (required)
- `banner-1544x500.png` - 1544x500px banner (required for retina)

### Screenshots
Place numbered screenshots here:
- `screenshot-1.png` - Enable conditional status in feed settings
- `screenshot-2.png` - Select form field to check
- `screenshot-3.png` - Choose statuses for TRUE/FALSE
- `screenshot-4.png` - Status mapping UI explanation
- `screenshot-5.png` - Example GDPR checkbox configuration

## Design Guidelines

- Use colors that match the Fluent ecosystem (blues and purples)
- Keep text minimal and readable
- Ensure images are optimized for web
- Screenshots should show actual plugin functionality

## Generating Assets

You can create these assets using:
- Canva (templates available)
- Adobe Photoshop/Illustrator
- Figma
- GIMP (free)
